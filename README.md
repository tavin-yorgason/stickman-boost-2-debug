## Stickman Boost 2 (debug edition)

### Features
| Keybind | Action                       |
| :---:   | :---:                        |
| `F1`    | Toggle slow fix              |
| `F2`    | Toggle hitboxes              |
| `F3`    | Toggle player placement mode |
| `F4`    | Create checkpoint            |
| `F5`    | Load checkpoint              |
| `-`     | Zoom out                     |
| `=`     | Zoom in                      |
| `0`     | Reset zoom                   |

### Quick start
Ensure you have Python installed, and run one of the `serve_` scripts.

Then, open `http://127.0.0.1:8000`.
